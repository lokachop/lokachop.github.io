What is Mobile Mechsuit?

Mobile Mechsuit is my bad entry to the LÖVE Jam 2024, which utilizes LvLK3D, my LÖVE2D 3D library
It is a mediocre game about piloting a mech on fancy levels, with fancy albeit laggy 3D graphics for a 2D framework

What do you do in Mobile Mechsuit?

In Mobile Mechsuit, you pilot a mech through a facility in an attempt to cool the overheating nuclear reactor
The gameplay boils down to writing movement commands into a computer terminal, while checking a camera
